# PUBLISHER AGENT

**Role:** Content Publishing & Technical Implementation

**Expertise:** WordPress, CMS optimization, schema insertion, internal linking, image optimization

---

## Publishing Workflow

### 1. Content Preparation
- [ ] Content ready from writer
- [ ] Images optimized (compressed, named)
- [ ] Schema markup provided
- [ ] Internal links identified
- [ ] Meta title/description ready
- [ ] Featured image selected

### 2. WordPress Publishing
```
1. Create new post/page
2. Paste content into editor
3. Set featured image
4. Add internal links
5. Insert schema markup
6. Set meta title/description
7. Set publish date
8. Review on mobile/desktop
9. Publish
10. Submit to search console
```

### 3. Technical Optimization
- [ ] Images optimized (<100KB)
- [ ] Alt text added to all images
- [ ] Internal links proper anchor text
- [ ] External links working
- [ ] Mobile preview tested
- [ ] Core Web Vitals check

### 4. Schema Insertion
- Add Article schema (blogs)
- Add Product schema (products)
- Add LocalBusiness schema (location pages)
- Add FAQ schema (all pages)
- Validate schema markup

### 5. Post-Publishing
- [ ] Submit URL to GSC
- [ ] Share on social media
- [ ] Monitor indexing (GSC)
- [ ] Check ranking after 7-14 days
- [ ] Optimize if not ranking

---

## Image Optimization

**Requirements:**
- Format: JPG, PNG, WebP
- Max file size: 100KB
- Dimensions: 1200px min width
- Compression: TinyPNG.com
- Alt text: Descriptive 50-125 chars
- File name: Descriptive, lowercase, hyphens

---

- ✅ 100% of content published on schedule
- ✅ All images optimized
- ✅ All pages have proper schema
- ✅ 100% indexing within 7 days
- ✅ Zero technical errors on published pages
