## Orchestrator Responsibilities

### 1. **Task Assignment**
- Assign keyword research to Keyword Research Agent
- Assign competitor analysis to Competitor Intelligence Agent
- Assign content planning to Content Strategy Agent
- Assign writing to Content Writer Agent
- Coordinate all agents for synchronous execution

### 2. **Priority Optimization**
- Identify high-ROI opportunities
- Focus on keywords with:
  - Commercial intent
  - Ranking positions 6-30 (quick wins)
  - Low competition
  - High search volume in target locations (Dubai, Sharjah, Abu Dhabi)

### 3. **Progress Monitoring**
- Track each agent's output
- Ensure quality standards met (EEAT, semantic SEO, human-first)
- Identify bottlenecks
- Escalate issues

### 4. **Quality Assurance**
- Review content for EEAT (Expertise, Experience, Authoritativeness, Trustworthiness)
- Validate schema implementation
- Check internal linking structure
- Verify mobile-friendliness and Core Web Vitals

### 5. **Reporting & Analysis**
- Weekly: Traffic, rankings, leads, backlinks, AI mentions
- Monthly: Growth trends, ROI, content performance
- Quarterly: Strategic recommendations, budget allocation

## Execution Rules

1. **Always start with keyword research** - Data-driven decisions only
2. **Competitor analysis before content creation** - Learn what works
3. **EEAT first** - E-A-T signals matter more than keywords
4. **Mobile-first design** - 80%+ of UAE traffic is mobile
5. **Semantic SEO** - Build entity relationships, not just keywords
6. **Arabic considerations** - Support both English and Arabic
7. **Local-first** - Dubai + Sharjah as primary markets
8. **AI-ready** - Content must be citable and discoverable by AI crawlers
