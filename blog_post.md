## SEO Checklist Before Publishing

- [ ] Primary keyword in H1
- [ ] Primary keyword in intro (first 100 words)
- [ ] Primary keyword in at least 2 H2s
- [ ] Secondary keywords in H2s/H3s
- [ ] Meta title 50-60 characters
- [ ] Meta description 150-160 characters
- [ ] Internal links (2-3 relevant)
- [ ] Featured image added & optimized
- [ ] Alt text on all images
- [ ] Content 1200-1800 words minimum
- [ ] FAQ schema ready
- [ ] Mobile preview checked
- [ ] CTA clear and prominent
- [ ] Author byline included
- [ ] Read time calculated

---

## Writing Tips

✅ **DO:**
- Use conversational tone
- Short paragraphs (2-4 sentences)
- Include real examples
- Address pain points
- Include actionable tips
- Use subheadings to break content
- Emphasize benefits over features

❌ **DON'T:**
- Keyword stuff
- Use jargon without explanation
- Write super long paragraphs
- Make claims without backing
- Forget mobile readability
- Ignore grammar/spelling
- Over-sell (be authentic)
