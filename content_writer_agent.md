## Content Types & Specifications

### Type 1: Blog Posts (Informational/Hybrid)

**Purpose:** Build topical authority, attract organic traffic, establish expertise

**Specifications:**
- **Length:** 1200-1800 words
- **Format:** Guide, tutorial, how-to, trend report
- **Structure:**
  ```
  H1 Title
  Brief intro (100-150 words)
  H2 Section 1 (250-350 words)
  H2 Section 2 (250-350 words)
  H2 Section 3 (250-350 words)
  FAQ Section (5-7 questions)
  CTA
  ```

**Includes:**
- ✅ Featured image (1200x800px minimum)
- ✅ 2-3 internal links (natural, relevant)
- ✅ FAQ schema markup
- ✅ Meta title (50-60 chars)
- ✅ Meta description (150-160 chars)
- ✅ Author byline: "Suhail Ahmed Gour, Tile Specialist"
- ✅ Read time indicator
- ✅ 2-3 relevant images throughout

**Example Topics:**
- "Complete Guide to Tile Installation"
- "Porcelain vs Ceramic Tiles: Which Should You Choose?"
- "Design Trends in Tile 2024"
- "Bathroom Tile Layout Ideas"
- "Tile Durability: What You Need to Know"

## Semantic SEO Requirements

### Entity Building
Mention related concepts naturally:
```
If writing about "Porcelain Tiles":
- Include: Ceramic, glazed, unglazed, durability, applications
- Link to: Related products, installation guides, comparisons
- Build: Relationships between concepts
```

### Semantic Variations
```
Primary: "porcelain tiles"
Variations:
- porcelain tile, tiles (porcelain)
- vitrified tile, ceramic porcelain
- porcelain ceramic, slab
- tile (porcelain), porcelain surface
```

### Context & Relationships
- Link complementary topics
- Reference related products
- Show knowledge connections
- Build topical clusters
- Support pillar pages

## FAQ Section Format

### Requirements for FAQ

**Structure:**
```
FAQ (Heading H2)

Q1: What is [main topic]?
A: [Clear, 50-100 word answer]

Q2: How does [product/service] work?
A: [Detailed, 100-150 word answer]

Q3: Which [product] should I choose?
A: [Comparative answer with recommendation]

Q4: What is the cost/price?
A: [Pricing guidance without exact numbers]

Q5: Can I see samples/schedule a visit?
A: [Call-to-action with contact method]
```

**Schema Requirements:**
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "Question text?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Answer text here"
      }
    }
  ]
}
```

## Meta Optimization

### Meta Title
- **Format:** [Primary Keyword] - [Benefit] | [Brand]
- **Length:** 50-60 characters
- **Examples:**
  - "Porcelain Tiles Dubai - Premium Quality | Volark"
  - "Bathroom Tile Ideas - Complete Design Guide"
  - "Ceramic vs Porcelain Tiles - Which Is Best?"

### Meta Description
- **Format:** [Benefit] + [Action] + [Uniqueness]
- **Length:** 150-160 characters
- **Examples:**
  - "Discover 20+ porcelain tile designs for Dubai homes. Expert specs, installation tips & prices. Request free samples now."
  - "Learn the difference between ceramic & porcelain tiles. Choose the right material for your project with our detailed comparison."
